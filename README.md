# Rest API + Spring MVC

## Для запуска клиента:
- скачать и установить nodeJS (https://nodejs.org/en/download/) 
- (выполнить в папке app-cliеnt) npm i
- (выполнить в папке app-cliеnt) ng serve

## Для запуска сервера:
- (выполнить в папке app-server) gradlew bootRun
